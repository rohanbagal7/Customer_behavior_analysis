# Customer_behavior_analysis
Data Analytics Project showcasing customer behavior analysis using python, sql, powerbi
This project demonstrates an end-to-end Data Analytics Workflow using Python, SQL, and Power BI.
The main goal of the project is to load raw data, clean and analyze it, extract business insights using SQL queries, and visualize results through an interactive dashboard and professional report.

The project covers:

Data Loading using Python
Exploratory Data Analysis (EDA)
Data Cleaning & Transformation
SQL Analysis using PostgreSQL, MySQL, and SQL Server
Power BI Dashboard Development
Report & Presentation Creation
📂 Dataset

The dataset contains business-related transactional data used for analysis and reporting.

Dataset Features
Customer Information
Product Details
Sales Transactions
Revenue & Profit Data
Order Status
Date & Time Information
